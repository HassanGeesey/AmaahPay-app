# Minimalist Dashboard

A clean, professional multi-column dashboard built with React and TypeScript, featuring a minimalist aesthetic with high information density.

## Features

- **Multi-column Layout**: Responsive grid-based card system
- **Activity Feed**: Real-time activity updates with user avatars and status indicators
- **Project Management**: Progress tracking, team avatars, and deadline management
- **File Management**: File listings with type icons and metadata
- **Quick Stats**: Key metrics at a glance
- **Responsive Design**: Optimized for desktop, tablet, and mobile

## Design Principles

- **Minimalist Aesthetic**: Clean, professional interface with subtle shadows and rounded corners
- **Light Theme**: White and soft gray backgrounds with soft accent colors
- **Typography**: Modern sans-serif fonts with clear visual hierarchy
- **Micro-interactions**: Smooth hover states and staggered animations
- **High Information Density**: Enterprise-ready but friendly interface

## Technology Stack

- React 19 with TypeScript
- Vite for development and building
- CSS3 with custom properties
- Responsive grid layout

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build

```bash
npm run build
```

## Components

- `Dashboard`: Main dashboard container
- Activity feed with user avatars and status indicators
- Project cards with progress bars and team avatars
- File listings with type-specific icons
- Stats grid for key metrics

## Styling

The dashboard uses custom CSS with CSS custom properties for:
- Consistent color scheme
- Responsive breakpoints
- Smooth transitions
- Component-specific styling
